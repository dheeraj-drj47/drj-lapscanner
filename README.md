# drj-lapscanner
Laptop Recovery Portal Prevention is better than loss. This repository hosts a lightweight, mobile-responsive web portal designed to help lost laptops find their way home.  By sticking a custom QR code onto your device, you provide anyone who finds it with an immediate way to contact you via:  Direct Phone/WhatsApp  Social Media: Instagram.
here is the direct link : https://drj-lapscanner.netlify.app/
